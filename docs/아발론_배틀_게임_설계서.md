## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| **장르** | 5:5 적진 점령 (AOS / MOBA) + RNG(운빨) |
| **플랫폼** | 모바일 / PC (크로스 플랫폼) |
| **시점** | 쿼터뷰 (줌인/줌아웃 지원) |
| **게임 목표** | 맵 양 끝에 위치한 적 본진의 **넥서스(코어)** 파괴 |
| **플레이 타임** | 20분 내외 / 20분 경과 시 **서든데스** 돌입 |
| **플레이어 수** | 유저 1명 + AI 9명 (아군 AI 4명 / 적 AI 5명) |

## 2. 핵심 게임 루프

```
게임 시작
  │
  ├─► 캐릭터 픽 (6종 중 1종 선택)
  │
  ├─► 인게임
  │     ├─ 라인전 & 정글링
  │     ├─ 미니언 처치 → 골드 획득
  │     ├─ 적 영웅 처치 → 골드 + 경험치
  │     ├─ 레벨업 → 랜덤 스탯 상승 (운빨!)
  │     ├─ 상점 → 아이템 구매 / 강화
  │     └─ 보스 처치 → 막대한 보상
  │
  ├─► 20분 경과 시 서든데스 크리쳐 등장
  │
  └─► 적 넥서스 파괴 → 승리 / 아군 넥서스 파괴 → 패배
```

### 핵심 차별점 요약
1. **극한의 운빨 성장** — 레벨 업마다 5종 스탯 중 1개가 완전 무작위 상승
2. **고도화 AI 동료/적** — 사람과 구분하기 힘든 수준의 판단 AI (9명 전원)
3. **오토 타겟팅 & 자동 평타** — 피지컬 피로도 최소화, 전술/스킬 타이밍에 집중
4. **20분 서든데스** — 거대 크리쳐가 모든 타워를 파괴하며 강제 한타 유발

## 3. 조작 시스템

### 3-1. 공통
- **게임 시작 순서:** [모바일/PC 환경 선택] → [영웅 1종 픽] → 게임 진입
- **미니맵:** 좌측 상단
- **상점 아이콘:** 우측 상단

### 3-2. PC 조작
| 입력 | 동작 |
|------|------|
| `W` `A` `S` `D` | 이동 |
| `O` | 스킬 1 사용 |
| `P` | 스킬 2 사용 |
| `우클릭` | 특정 지점으로 이동 지시 |

### 3-3. 모바일 조작
| 입력 | 동작 |
|------|------|
| 좌측 가상 조이스틱 (V-Pad) | 이동 |
| 우측 버튼 (2개) | 스킬 1 / 스킬 2 |
- 강제 **가로 모드** (Landscape)
- 하단부 UI 영역 완전히 비워둠 (Clean UI)
- 주사율 차이 보정 필수 (부드러운 프레임)

### 3-4. 자동 전투 시스템
- **자동 평타:** 이동하지 않거나 사거리 내에 적이 있으면 자동으로 기본 공격 출력
- **타겟 우선순위:**
  1. 사거리 내 적 영웅
  2. 나를 공격 중인 적
  3. 타워
  4. 미니언

- **스킬 타겟팅:** 즉시 발동 스킬을 제외하고 타겟 우선순위에 따라 자동 시전

### 3-5. 비전투 회복
- 공격/스킬 사용 없이 **5초** 대기 시 (이동 무관) → HP 지속 회복 시작
- 회복량: `maxHp × 5% × 초당`

## 5. 성장 시스템

### 5-1. 경험치 & 레벨업

| 항목 | 내용 |
|------|------|
| 레벨 제한 | **없음** (무한 레벨업) |
| 기본 경험치 필요량 | 100 (레벨업마다 ×1.2 증가) |
| 적 영웅 처치 시 경험치 | 50 |
| 경험치 획득처 | 미니언 처치, 적 영웅 처치, 정글 몬스터 처치 |

### 5-2. 랜덤 스탯 상승 (핵심 운빨 요소)

레벨업 시 아래 4가지 스탯 중 **1개가 완전 무작위**로 선택되어 상승:

| 스탯 | 상승량 |
|------|--------|
| 공격력 (attackDamage) | +5 |
| 최대 체력 (maxHp) | +50 (현재 HP도 동량 증가) |
| 이동 속도 (moveSpeed) | +0.5 |
| 공격 속도 (attackSpeed) | +0.2 |

> 💡 매 판마다 같은 캐릭터라도 완전히 다른 역할로 성장할 수 있음.  
> 예: 토르가 공격속도 특화 → 빠른 근접 딜러 / 이속 특화 → 추격/도주 특화

### 5-3. 부활 시간

```
부활 시간(초) = 5 + (현재 레벨 × 2)
```

- 레벨 1: 7초
- 레벨 5: 15초
- 레벨 10: 25초

### 8-3. 기본 아이템 목록 (100골드)

| 아이템 | 타입 | Lv.1 효과 | 강화 배율 |
|--------|------|-----------|-----------|
| **공격력 강화** | `attack` | 공격력 +15 (mult=1.5) | `10 × (1 + lv × 0.5)` |
| **공격 속도 강화** | `speed` | 공격속도 +0.15 | `0.1 × (1 + lv × 0.5)` |
| **체력 강화** | `hp` | 최대HP +150, 현재HP +150 | `100 × (1 + lv × 0.5)` |
| **이동 속도 강화** | `move` | 이동속도 +0.75 | `0.5 × (1 + lv × 0.5)` |
| **크리티컬 강화** | `crit` | 크리티컬 확률 +7.5% | `0.05 × (1 + lv × 0.5)` |
| **흡혈** | `lifesteal` | 흡혈률 +15% | `0.1 × (1 + lv × 0.5)` |
| **피해 반사 (가시방패)** | `reflect` | 피해 반사 +15% | `0.1 × (1 + lv × 0.5)` |
| **화염 검 (도트 피해)** | `burn` | 평타 시 도트 피해 +7.5 (3틱) | `5 × (1 + lv × 0.5)` |
| **기절 무기** | `stun_weapon` | 평타 기절 확률 +7.5% (1초) | `0.05 × (1 + lv × 0.5)` |

> **강화 배율 공식:** `mult = 1 + (level × 0.5)`  
> Lv.1=1.5, Lv.2=2.0, Lv.3=2.5, Lv.4=3.0 ... (기하급수적 증가)

| +7 | 30% | 골드만 소모 |

| +9 | 10% | 골드만 소모 |

### 8-5. 에픽 아이템 (조합 아이템)

기본 아이템 여러 개를 소모하여 조합. AI도 에픽 아이템 조합을 시도함.

#### ⚔️ 무한의 대검 (Infinity Edge)
- **재료:** 공격력 + 크리티컬 아이템 각 1개
- **조합 비용:** 150 골드
- **효과:** 공격력 +80, 크리티컬 확률 +50%

#### 🛡️ 가시 갑옷 (Thornmail)
- **재료:** 체력 + 피해반사 아이템 각 1개
- **조합 비용:** 150 골드
- **효과:** 최대HP +500, 피해 반사 +50%

#### 🗡️ 몰락한 왕의 검 (Blade of the Ruined King)
- **재료:** 공격력 + 공격속도 + 흡혈 아이템 각 1개
- **조합 비용:** 200 골드
- **효과:** 공격력 +40, 공격속도 +0.5, 흡혈 +20%, **현재 적 HP의 8% 추가 피해** (평타마다)

#### ❤️ 워모그의 심장 (Warmog's Heart)
- **재료:** 체력 아이템 Lv.3 이상 보유 필요
- **조합 비용:** 150 골드
- **효과:** 최대HP +1,000, **비전투 시 HP 회복 속도 2배** (기존 5% → 10%/초)

## 9. 전투 상세 규칙

### 9-1. 크리티컬 히트
- 크리티컬 발생 시 **피해량 2배**
- 기본 크리티컬 확률: 캐릭터마다 다름 (궁수 기본 10%)
- 플레이어가 적 처치 시 화면에 데미지 텍스트 표시 (크리티컬은 별도 표시)

### 9-2. 기절 (Stun)
- 기절 상태: 행동 전면 불가
- 기절 지속 시간: 스킬 기절 **2초**, 기절 무기 **1초**
- 기절 중에는 HP 회복도 되지 않음

### 9-3. 도트 피해 (화염검)
- 평타 1회당 `burnDamage` 만큼 **1초 간격 3회** 추가 피해
- 건물(타워, 넥서스)에는 도트 피해 **미적용**

### 9-4. 피해 반사 (가시방패/Thornmail)
- 받은 피해의 `reflect` 비율만큼 **공격자에게 반사**
- 건물의 공격에는 반사 **미적용** (건물은 반사 피해를 받지 않음)

### 9-5. 현재 체력 비례 피해 (몰락한 왕의 검)
- 평타 적중 시 **대상의 현재 HP × 8%** 추가 피해

### 9-6. 부활
- 사망 후 일정 시간 대기 → 자동 부활
- 부활 위치: 아군 본진 (넥서스) 근처
- 부활 시 HP 완전 회복

## 10. AI 행동 로직

유저 개입(핑, 오더) 없이 완전 자율로 작동. AI도 유저와 동일하게 랜덤 스탯 레벨업 및 상점 아이템 구매 진행.

### 10-1. 상황 판단 (HP 기반)

| 조건 | 행동 |
|------|------|
| HP ≤ 30% | 후퇴 모드 진입 (isRetreating = true) |
| HP ≥ 80% | 후퇴 모드 해제, 전투 복귀 |

**후퇴 시 행동:** 가장 가까운 **아군 건물**을 향해 이동 (타워/넥서스)

### 10-2. 타겟 선정 (AI)

AI의 타겟 점수 계산:
```
score = -거리 + (1 - HP비율) × 50

적 영웅이면: score += 100
건물이면:    score += 50
인식 범위: 30 이내
```

→ 가장 점수 높은 대상을 타겟으로 선정 (**막타 우선 로직** 내장)

### 10-3. 합류 & 한타 판단

- 맵 어딘가에서 전투 발생 시 → 동선 판단 후 지원 합류
- 2명 이상이 뭉칠 경우 → 전력 계산 후 돌관/수비 결정

### 10-4. 다양한 전술

- 라인이 비면 → 정글(필드 몹) 파밍 (`HP > 50%` 조건, 5% 확률로 트리거)
- 혼자 있는 적 영웅 발견 시 → 암살 시도
- 도망치는 척하며 아군 타워 쪽으로 유인 후 역공

### 10-5. 타워 다이브 금지

- 적 타워 사거리 내로의 무지성 돌진 전면 차단

### 10-6. AI 아이템 구매 로직

- **15초마다** 아이템 구매 시도
- 골드 150 이상: 에픽 아이템 조합 먼저 시도 → 실패 시 기본 아이템 구매
- 골드 100~149: 기본 아이템 랜덤 구매
  - 구매 대상: 공격, 공속, 체력, 이속, 크리 중 랜덤

### 10-7. 정글 몬스터 AI

- 캠프 영역 안에서만 활동
- 5초간 비전투 → 타겟 초기화 + 캠프로 귀환
- 영역 밖 원거리 공격 감지 시 → 빠른 속도로 돌진
