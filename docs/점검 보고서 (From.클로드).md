### 🔴 [심각] 쿨타임 최솟값이 너무 낮음

```
실제 최소 쿨타임 = max(2초, baseCd - level보정) * max(0.3, 1 - CDR)
                 = 2초 * 0.3 = 0.6초
```

즉 **초당 1.67회** 스킬 발동이 가능합니다.  
현자의 지팡이 1개(15% CDR)를 7강화(×4.5배) 하면 CDR = 67.5% → 캡 55% 초과.  
여기에 `haste_art` 패시브 3레벨(30% CDR)이 추가되면 → **CDR 85% → 실제 적용 55%**, `Math.max(0.3)`으로 막히지만 여전히 최소 0.6초.

| 구분 | 현재 값 | 권장 값 |
|---|---|---|
| CDR 소스 캡 (`this.cdr`) | 55% | 40% |
| 최종 적용 CD 배수 | `Math.max(0.3)` = 70% 감소 | `Math.max(0.5)` = 50% 감소 |
| 절대 최소 쿨타임 | 0.6초 | 1.5초 |
